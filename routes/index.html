SDMX API for EViews


This API allows you to retrieve an INSEE series by its idbank or a dataset by its id.

How to use it ?

For a Timeseries :

Input :

  • http://sdmx.herokuapp.com/series/idbank?startPeriod=startPeriod
  • http://sdmx.herokuapp.com/series/idbank?lastNObservations=lastNObservations
Example : http://sdmx.herokuapp.com/series/000436387?startPeriod=2016

Output :

Dates000436387
 Démographie - Population au début du mois - France métropolitaine
2016-0164513
2016-0264528
2016-0364543
2016-0464554
2016-0564570

Multiple timeseries

You can retrieve multiple timeseries at a time (they should have the same frequency though) by using :

http://sdmx.herokuapp.com/series/idbank1+idbank2?startPeriod=startPeriod

For a DataSet :

Input :

  • http://sdmx.herokuapp.com/dataset/dataset_id?freq=frequency
Example : http://sdmx.herokuapp.com/dataset/IPCH-2015-FR-COICOP?freq=M

Know Issues

Using the filter ?lastNObservations= when retrieving multiple series or a dataset is a bad idea if series do not belong to the same time period.
