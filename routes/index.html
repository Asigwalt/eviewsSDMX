SDMX API for EViews


This API allows you to retrieve an INSEE series by its idbank.

How to use it ?

Input :

  • https://sdmx.herokuapp.com/idbank?startPeriod=startPeriod
  • https://sdmx.herokuapp.com/idbank?lastNObservations=lastNObservations
Example : https://sdmx.herokuapp.com/000436387?startPeriod=2016

Output :

Dates000436387
 Démographie - Population au début du mois - France métropolitaine
2016-0164513
2016-0264528
2016-0364543
2016-0464554
2016-0564570
