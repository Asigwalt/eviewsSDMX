ERROR 408 : Request has timed out.

Your request has timed out, here is why:
  • The dataset was too big

    The dataset you requested was too big and it takes too long to send it back. You should cut it according to one or multiple dimensions using filters.
    See here how to do so.
    You should also see the dimensions list at : http://sdmx.herokuapp.com/PROVIDER/dataflow/DATASET_ID (the capital letters are to be replaced by the actual value according to the provider and dataset id you want to retrieve data from).


  • The request was wrongly specified.

    You made a mistake specifying the request and the server can't find what you mean. See here how to use the filters.
